The posters for the PeP-Bachelor Kolloqium.

# Building
To produce the output to `build/main.pdf` simply run
```sh
make
```

To use continuous building (i.e. while working on the document) use 
```sh
make work
```

# Dependencies 
You need 
- texlive 22
- make
